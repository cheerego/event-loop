# Micro Task 与 Macro Task


* nextTick 在哪里执行，属于事件循环的那个阶段呢？
```javascript
process.nextTick(function(){
    console.log(7);
});

new Promise(function(resolve){
    console.log(3);
    resolve();
    console.log(4);
}).then(function(){
    console.log(5);
});

process.nextTick(function(){
    console.log(8);
});

```
